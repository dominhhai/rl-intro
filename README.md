# Install
```
$ pip install virutalenv
$ virutalenv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
```

# Usage
```
$ source .venv/bin/activate
$ jupyter notebook
```

# Halt
```
$ deactivate
```
